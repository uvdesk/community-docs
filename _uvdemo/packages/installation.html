Steps To Install a Package in Open Source Helpdesk Ticketing System

Step 1 -

Browse to the apps directory of your helpdesk installation

Step 2 -

Inside the apps directory put your package source code

Step 3 -

Now, Go back to your helpdesk root directory and run the following commands:

Package Configuration:

php bin/console uvdesk_extensions:configure-extensions

Assets Installation:

php bin/console assets:install

Clear Cache:

php bin/console cache:clear

Now, The package will be successfully installed into your helpdesk installation
