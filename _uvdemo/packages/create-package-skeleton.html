Creating a package skeleton

Step 1 -

Browse to the root directory of your helpdesk installation and run the following commands:

Step 2 -

Execute the following command to generate the skeleton:

php bin/console uvdesk_extensions:create-package-skeleton

Step 3 -

Enter your package vendor's name (The name must in PascalCase)

Step 4 -

Enter the package name (The name must in PascalCase)

Step 5 -

Select the package type from the given list

Step 6 -

Add authors to the package

Step 7 -

Choose whether your package will be configurable or not.

Step 8 -

Now, the package skeleton will have successfully generated.

Step 9 -

Choose whether to configure your package or not.

Step 10 -

If you did not face any errors and/or exceptions, your new package skeleton will have successfully created and configured.
